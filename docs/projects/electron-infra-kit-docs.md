# electron-infra-kit-docs

> electron-infra-kit 的完整文档站点，提供详细的 API 文档、使用指南和最佳实践

## 项目简介

electron-infra-kit-docs 是 electron-infra-kit 框架的官方文档站点，使用 VitePress 构建。文档涵盖了框架的所有功能模块，帮助开发者快速上手并深入理解框架的设计理念。

### 📖 完整的 API 文档

- 窗口管理器 API
- IPC 路由 API
- 消息总线 API
- 配置管理器 API
- 类型定义和接口说明

### 📚 使用指南和最佳实践

- 快速开始指南
- 核心概念讲解
- 常见使用场景

- 故障排查指南

### 💡 示例代码和教程

- 基础示例
- 进阶用法
- 实战案例
- 常见问题解答
- 迁移指南

## 技术栈

- **VitePress** - 现代化的文档站点生成器
- **Markdown** - 简洁的文档编写格式
- **Vue 3** - 自定义组件支持

## 文档特色

- 🚀 **快速加载** - 基于 Vite 的极速构建
- 🎨 **美观界面** - 现代化的设计风格

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/chunhaofen/electron-infra-kit-docs.git

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建文档
npm run build
```
