# 我的项目

这里展示了我维护的开源项目，主要专注于 Electron 桌面应用开发的基础设施建设。

<ProjectGrid />

## 项目系列

### electron-infra-kit 系列

这是一套完整的 Electron 应用基础设施解决方案，包括：

- **[electron-infra-kit](/projects/electron-infra-kit)** - 核心工具包，提供窗口管理、IPC 路由、消息总线等功能
- **[electron-infra-kit-docs](/projects/electron-infra-kit-docs)** - 完整的文档站点，包含 API 文档和使用指南
- **[electron-infra-showcase](/projects/electron-infra-showcase)** - 官方示例项目，展示框架的实际应用

## 技术特点

- ⚡ **高性能** - 优化的架构设计，确保应用流畅运行
- 🔒 **类型安全** - 完整的 TypeScript 支持
- 📦 **开箱即用** - 简单的 API 设计，快速上手
- 🔧 **可扩展** - 灵活的插件系统，支持自定义扩展
- 📚 **文档完善** - 详细的文档和示例代码
